# Shopping App

Une application de panier d'achat construite avec Angular.

## Caractéristiques

- Affichage des produits
- Gestion du panier
- Interface utilisateur responsive

## Démarrage

### Installation

```bash
npm install
```

### Exécution

```bash
ng serve
```

Ouvrez [http://localhost:4200](http://localhost:4200) dans votre navigateur.

### Build

```bash
ng build
```

## Structure du projet

- `src/app/components/` - Composants réutilisables (header, product-list, cart-dialog)
- `src/app/services/` - Services (products.service, cart.service)
- `src/app/models/` - Modèles de données

## Technologies

- Angular
- TypeScript
- SCSS
- Angular Material (Dialog)
